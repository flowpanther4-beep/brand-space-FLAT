# Brand Space™ — Retro Pixel Wall (Next.js + Tailwind + Vercel)

Front-end listo para deploy. Estilo 2005 con detalles 2025.

## Scripts
- `npm run dev` — desarrollo local
- `npm run build` — build producción
- `npm start` — servir build

## Deploy en Vercel
1. Sube este proyecto a un repo de GitHub (Upload files).
2. En Vercel → New Project → Importa el repo → Deploy.

## Estructura
- `src/app` — App Router (layout, page, estilos globales)
- `src/components` — Header, Menu, PixelGrid, PixelBlock
